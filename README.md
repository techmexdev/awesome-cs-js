- Syntax
- Loops, Conditionals, Basic Data Structures
- Intermediate Learning
- Functions, Closures, Objects, and Prototypes
- Best Practices
- Advanced Topics
- Higher Order Functions
- Troubleshooting
- Debugging
- Further Learning
- Coding Challenges
- JS Fundamentals Workshop
- Advanced Javascript Knowledge
